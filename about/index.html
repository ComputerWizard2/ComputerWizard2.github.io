项目一：橘子电子商城项目

开发环境： Eclipse + Tomcat8.0 + Mysql5 + SVN + Maven

项目周期：2017.08.01-2017.09.01

软件架构： Spring + SpringMVC + MyBatis+ Ajax + Jquery + MySQL + JSTL/EL

项目描述：

  • 1、这个项目是和同学一起开发的，使用到了ssm的框架，Spring来做容器装载前端的页面和后mybatis,使用操作数据层
  • 2、本项目分为后台管理以及前台的用户页面两部分。后台管理包括一些商品、商品分类、商品属性、活动、广告、优惠券以及由超级管理员分配的角色和权限管理。前台是面向用户的，包括商品展示、详 情、评论、购买以及用户自己的订单、购物车、优惠券等一些信息。

责任描述：

  • 1、参与需求分析以及数据库的设计
  • 2、后台方面，主要负责优惠券、活动以及广告管理；前台方面，主要负责商品分类列表、商品详情、评论、购物车、提交订单到支付等功能

技术描述：

  • 1，使用SSM作为底层框架，SpringMVC处理访问请求，调用业务代码并反馈，MyBatis使得数据的访问与操作更加强简易，Spring管理各架构层的bean,实现自动注入以及数据源的配置;
  • 2，使用github上的开源插件pageHelper进行分页操作。
  • 3，使用DOM操作处理一些页面的信息
  • 4，使用ajax实现局部页面的数据刷新
    当有户量激增的时候，我们应该考虑如何进行负载均衡，在服务器上面使用nginx搭建服务器的集群

项目三： 爱电影

开发环境： Eclipse + Tomcat8.0 + mysql+ SVN + Maven

项目周期：2016.11.12-2016.11.28

软件架构： Springboot+nginx+redis

项目描述：

  • 1、这个项目是自己对java爬虫的兴趣来完成的，在这个项目主要用到了jsoup来解析html,httpclient用于模拟用户发起请求，得到想要的html通过jsoup进行解析获相关的数据存入到数据库
  • 2、我们设计好页面，我们考虑到用户量会激增，所以我们采用nginx进行负载均衡，通过redix的NoSql缓解数据库的压力，将数据库分为主从库，主库进行相关的写入操作，从库进行相关的查询的操作，将项目的访问的性能进行优化
  • 3、本项目的全向主要分为两个模块：一个模块进行到相关的网站上进行数据的爬取解析，另一个模块对用来展示爬取的数据

责任描述：

  • 1、通过自己搭建java的jar，以及实现爬虫的爬取数据，主要使用jsoup来进行html 页面的解析，使用httpclient 进行模拟浏览器的请求，通过解析出来的数据存储到数据库中，供自身网站的使用。
    通过数据库的数据将爬取到数据放入显示
    技术描述：

  • 1，springboot进行开发，利用jpa对于数据的访问，jpa可以自动的生成SQL语句进行查找，MyBatis实现dao层访问数据库。利用mybatis自动生成工具生成javabean和dao层。

  • 2，自己封装了一个page类进行分页操作,进行电影页面的分页操作
    对于freemark进行页面的静态化处理
