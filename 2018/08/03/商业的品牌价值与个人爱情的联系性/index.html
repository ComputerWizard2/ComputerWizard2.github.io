商业的品牌价值与个人爱情的联系性

发表于 2018-08-03 | |
字数统计: | 阅读时长 ≈
  • 我们是否应该去追求自己喜欢的女孩：
    • 我想抛出这个观点，或者简介的说我们应该怎么去爱一个人，觉的自己对爱情真的是蒙蒙动动，并非那么真切，突然学习商品中的品牌价值，观看了张国樣先生的TED 的演讲，真的很受感触，彻底明白了，我们的商业帝国和我们个人自身的发展是差不多的。
    • 首先作为一个自主领导品牌，整个公司不可能是通过乞求或者说是追求顾客来进行消费，例如苹果，华为的等自主品牌的商业帝国，每诞生一款新品，都有一些追捧者去追求，甚至不惜排队购买。我们谈到苹果，不由自主的想到他的一些标签，软硬兼施，铝镁合金，简直就是一键艺术品，我也很喜欢苹果的品牌价值。就是这样，他不缺乏追求者，而带给我们思考的是，我们将它类比与我们自身，当我们自身是否具有这种品牌效应，我们又怎么恐惧我们缺乏追求者呢？
    • 我们谈追求，首先追求这两个字就给我们带入了误区，我们分开来看，求这个字我们可以理解为哀求，乞求，他给我们带入的就是注定两个人的地位是不平等的，两个人对于爱情的付出和回报注定就是不等价的，《东京爱情故事》里的莉香为丸子付出了那么多，结果换来的还是丸子和关口在一起，有些人将爱情是为生命的全部，他们就是敢爱敢恨，勇于追求自己爱情，哪怕付出再多也不后悔，而有些人将爱情视为生命的一部分，注定平淡无谓，不会轻易表露，当时看这个电影，我真的被这个热情，善良的姑娘打动了，甚至到电影的最后为她抹了一把眼泪。我们首先要摆正一个正确的自我价值的意识，我们不应该考虑的是我应该怎么让被追求者开心，快乐，感动，最后他因为感动而嫁给我，这条路注定是死路，就算能答应你，未来的婚姻也不会幸福，因为生活毕竟就是相互包容，而只有一方的无休止的包容，而另一方无休止的作妖只会导致婚姻的破裂。其次我们来分析一下，这个追字，我可以通过一个生活的例子来解释一下，还原一下各位心里，我曾经去超市买东西，正好赶上红酒特卖，我抱着试一试的态度去品尝，感觉味道还不错，就想买一瓶回家，喝了以后我又悄悄问售货员多少钱一瓶，他回答说128，我当时想买了，一摸兜，恰好钱包落在家了，就默不作声，毕竟拿人手短，可就在这个时候，他说：我这还有一瓶1280的要不给您打开尝尝，我放下酒杯连忙推脱掉，我说我还有别的事情，您先忙。其实这和我们追一个女孩不也是这样吗？起初我们送她一个简单的生日礼物，她品尝了一下，当她和我们疏远的时候，我们奉上更加贵重的礼物，在他不了解你的情况下，这样只会将她吓跑，当然这里也有喜欢到超市喜欢蹭吃蹭喝的消费者，毕竟我们这个社会人与人的价值关是不同的，素质也是不同的。所以以上我论述追求就是一种错误的方式。这也是我血的经验与教训，所得到的结果。
    • 我们应该采用采用何种方式呢，首先我们在创建一个商业帝国，列如：INtel或者苹果，他们都要建立自我的品牌价值，通过广大消费者购买商品，进行二次广播辐射，对产品进行思维固化，恰如克劳次的《重新定位》所描述的，要是你的产品逐渐的思维的固化在消费者的心里，你的产品才会得到市场的认可。同样我们在追求一个女孩的时候，我们最好就是提高我们自我品牌的价值，而不是花着时间来都妹子开心，而是价值，创造我们的价值。其次我们要传递价值，有些人总喜欢把自身的价值藏着掖着，我觉的根本没有必要，在这个世界上我们生命的意义就是在创造我们的价值，这个时候我们为什么不将我们的价值进行传递，寻找和我知趣相投的人，那样我们至少不会孤独。最后我们要建立连接，和我有相同价值关，有相同爱好的人建立联系，这样你的品牌效应才会扩大。最后才会扩展为你的商业帝国。你还会缺乏对你的追求者吗？
    • 所以我在最后总结一下，我们不需要去追求，我们要做的长期的去提升自我价值，创造自己的品牌效应，传递我们的价值，和我们志趣相投的人建立联系。最后我们可以收获我们的人生幸福。谢谢。
