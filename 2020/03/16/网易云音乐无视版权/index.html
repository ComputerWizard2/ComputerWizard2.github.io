网易云音乐无视版权

发表于 2020-03-16 | |
字数统计: | 阅读时长 ≈

  • 重要通知：因官方维权，所以该软件将停止更新，请大家支持正版！

  • 想必大家在网易云上经常看到这个吧

  • 作为国内音乐行业大IP,大流量的音乐平台，网易云很多歌曲却受到版权的限制，搜索周董的歌曲，没有几个能听的,最近看到一些小伙伴通过QX来破解网易云的灰色歌曲，心里痒痒，特别好奇这些小伙伴是如何破解的呢？

  • 所以在google上面search 一下，真的找到了源头

  • 源头就是github以为大佬发起的来的开源的node.Js脚本

  • 原理：就是通过网易云的灰色歌曲才用其他平台进行补充，这样就是运行着网易的APP，其实是qq，酷我，酷狗等多家音乐平台。内容很全。这样就理解了QX在这个过程就是做了一个代理分流的策略。
